## MKNote

A sidebar notebook.

![intro](https://user-images.githubusercontent.com/5998498/210298425-4f5b16cf-4bca-4bbe-9e9d-f2f455282b5f.png)

Features:

- Side-Bar Notes
- New File / New Group
- Move / Copy / Paste / Delete / Drag / Rename
- Reveal in finder
- Open in new window
- Git bind & sync
- All ext supports
